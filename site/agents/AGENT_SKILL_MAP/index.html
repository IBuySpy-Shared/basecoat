    Getting Started¶

    For New Users¶

    1. Find an Agent — Browse Agents by Discipline to find an agent for your role
    2. Explore Compatible Skills — Check Agent-Skill Mappings for complementary skills
    3. Read Skill Documentation — Each skill in skills/*/SKILL.md contains detailed patterns and examples
    4. Use Agent Commands — Access agents through GitHub Copilot (VS Code, Cursor, Windsurf, Claude Code)

    For Developers Extending Base Coat¶

    Adding a New Skill: 1. Create skills/{skill-name}/SKILL.md with Agent Skills spec frontmatter 2. Include name, title, description, compatibility, metadata, allowed-tools 3. Add domain classification in metadata.domain 4. Update this index with new skill entry

    Adding a New Agent: 1. Create agents/{agent-name}.agent.md 2. Include Agent Skills spec frontmatter 3. Define agent workflows and capabilities 4. Update this index with new agent entry

    Linking Agents to Skills: Use the compatibility field in skill frontmatter: 

    compatibility: ["agent:backend-dev", "agent:data-tier"]

    Integration Paths¶

    VS Code Copilot¶

    • Agents: Use / to access agents (e.g., /backend-dev)
    • Skills: Auto-discovered from .agents/skills/ per Agent Skills specification

    Cursor / Windsurf¶

    • Access via .agents/skills/ directory for skill discovery
    • Full spec compliance enabled via sync.ps1 / sync.sh

    Claude Code¶

    • Agents: Native support via Agent Skills spec
    • Skills: Discovered from .agents/skills/ structure
