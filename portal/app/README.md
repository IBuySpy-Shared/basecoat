# Companion App Workspace

This folder is the organized companion app workspace.

Phase 1 introduces structure and planning artifacts without forcing immediate runtime path breaks.

## Layout

- `backend/` — companion app API/service boundary
- `dashboard/` — companion app web interface boundary
- `db/` — schema and data operations boundary
- `iac/` — companion app infrastructure modules and parameters
